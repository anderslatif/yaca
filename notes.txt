Socket.io
----------------------------------------------------------------------------------------

A global object with namespaces as key and member limits as key-value pair will be made available globally so that the member limit can be looked up.

This function will loop through the rooms and see if there are rooms where the amount of sockets is below the member limit. The first it finds that fulfills this case it will let the socket join that room. If all the rooms are full I should create a new room.

On disconnect I should get the socketId and have them leave the room.
